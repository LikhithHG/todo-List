A Personal Task Manager App will allow users to: 
✅ Add Tasks – Users can add new tasks.
✅ Edit Tasks – Users can modify existing tasks.
✅ Delete Tasks – Users can remove completed or unwanted tasks.
✅ Mark Tasks as Completed – Checkbox to mark tasks as "Done."
✅ Task Prioritization – Add High, Medium, Low priority tags.
✅ Task Filtering – View All, Active, or Completed tasks.
✅ Persistent Storage – Save tasks using localStorage so they remain after refresh.
✅ Dark Mode (Optional) – Allow users to switch between light/dark mode.
